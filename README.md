# Vinarija i OPG Dine

Moderni web dizajn za obiteljsku vinariju Vinarija i OPG Dine iz Blata na Korčuli.

## Opis Projekta
Web stranica je dizajnirana s premium estetikom, fokusirajući se na tradiciju i kvalitetu vina Dine. Projekt je nastao kao visokokvalitetna replika modernih vinskih stranica, prilagođena lokalnom brendu.

## Tehnologije
- **HTML5** - Semantička struktura
- **Vanilla CSS3** - Moderni dizajn, Flexbox/Grid, animacije
- **JavaScript** - Interaktivnost, animacije brojača, mobilni izbornik
- **Google Fonts** - Playfair Display (naslovi), Lato (tekst), Great Vibes (potpis)
- **Font Awesome** - Ikone

## Struktura
- `index.html` - Glavna stranica
- `index.css` - Svi stilovi i responzivnost
- `script.js` - Dinamički efekti i navigacija
